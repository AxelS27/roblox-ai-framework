# UI/UX Style Guide & Component Index

This folder registers all UI/UX screen mockups, responsive layouts, style tokens, and reusable component blueprints.

## UI Folder Structure
* **[design-system.md](file:///d:/Experiments/Roblox%20AI%20Framework/.agents/GDD/ui_ux/design-system.md)**: Color systems (HSL/Hex), typography, and safe zone guidelines.
* **[components/](file:///d:/Experiments/Roblox%20AI%20Framework/.agents/GDD/ui_ux/components/)**: Reusable UI component specs and instance blueprints:
  * **[button.md](file:///d:/Experiments/Roblox%20AI%20Framework/.agents/GDD/ui_ux/components/button.md)**: Primary, Secondary, Ghost, Danger, and Loading button states.
  * **[dialog.md](file:///d:/Experiments/Roblox%20AI%20Framework/.agents/GDD/ui_ux/components/dialog.md)**: Modal popups, backdrop blurs, and confirmation dialogs.
  * **[hotbar.md](file:///d:/Experiments/Roblox%20AI%20Framework/.agents/GDD/ui_ux/components/hotbar.md)**: Item hotbars, inventory slots, equipment selection, and cooldown sweeps.
  * **[progress-bar.md](file:///d:/Experiments/Roblox%20AI%20Framework/.agents/GDD/ui_ux/components/progress-bar.md)**: Health, Stamina, XP, and lag fill progress bars.
  * **[card.md](file:///d:/Experiments/Roblox%20AI%20Framework/.agents/GDD/ui_ux/components/card.md)**: Shop item cards, inventory item tiles, and rarity borders.
  * **[input-field.md](file:///d:/Experiments/Roblox%20AI%20Framework/.agents/GDD/ui_ux/components/input-field.md)**: Search bars, promo input textboxes, and clear actions.
  * **[slider.md](file:///d:/Experiments/Roblox%20AI%20Framework/.agents/GDD/ui_ux/components/slider.md)**: Volume, FOV, and RGB color picker sliders.
  * **[tab-bar.md](file:///d:/Experiments/Roblox%20AI%20Framework/.agents/GDD/ui_ux/components/tab-bar.md)**: Category selector tabs and indicator line tweens.
  * **[tooltip.md](file:///d:/Experiments/Roblox%20AI%20Framework/.agents/GDD/ui_ux/components/tooltip.md)**: Hover stats cards and skill description tooltips.
  * **[player-card.md](file:///d:/Experiments/Roblox%20AI%20Framework/.agents/GDD/ui_ux/components/player-card.md)**: Player profile popups, avatar thumbnails, and detailed stats cards.
  * **[leaderboard.md](file:///d:/Experiments/Roblox%20AI%20Framework/.agents/GDD/ui_ux/components/leaderboard.md)**: Custom leaderboard player lists, rank slots, and active leaderstats columns.
* **[templates/screen-spec.md](file:///d:/Experiments/Roblox%20AI%20Framework/.agents/GDD/ui_ux/templates/screen-spec.md)**: Template for specifying individual UI Screen GUIs.
